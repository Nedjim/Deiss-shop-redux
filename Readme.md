# Deiss-Shop - Redux

Lien du site: [https://deiss-shop.herokuapp.com/](https://deiss-shop.herokuapp.com/)
## Technologies
- Node.js
- Express
- MongoDB / Mongolab
- Mongolab
- React.js / Redux
- Webpack
- Sass
- Heroku

## Exécution
```
$ npm install
$ npm start
```

## Routes 
- http://localhost:8080/
- http://localhost:8080/boutique
- http://localhost:8080/auth
- http://localhost:8080/create
- http://localhost:8080/admin/articles
